# DAW's plotting examples

This repo contains two sets of plotting examples I created for my former research groups.

### MME55 benchmarking: MATLAB

These examples were created for Dr Lars Goerigk's group (University of Melbourne) in Jan 2024 and are based on the DFT tests from my [MME55 benchmarking paper](https://doi.org/10.1021/acs.jctc.3c00558).
They demonstrate the custom functions I made for extracting specific values from the data table, and how to make the group's most commonly used plot types in MATLAB.


### Chorismate mutase MD: jupyter notebook/matplotlib

These examples were created for Dr Nate DeYonker's group (University of Memphis) in Nov 2025 and use RMSDs and probe/arpeggio residue interaction networks calculated for each frame of the MD simulation from the group's [2024 PCCP paper](https://doi.org/10.1039/D3CP06100K). They were designed to be a simple demonstration of jupyter notebooks/pandas/matplotlib (and python more generally) for a series of group meetings introducing the grad students to computational enzymology.
