# Function descriptions
### Examples are based on the MME55 examples and need that data loaded

- custom_legend: creates a legend from a set of colours and labels, independent of whatever else is actually plotted on the axes (generalised from how I did it in example 8)
- scale_lgd_icons: scale scatter plot legend icons to [default 0.8] of legend text size. can also change outline of circle markers to look fancy. demonstrated in examples of custom_legend and setdatatips.
- setdatatips: select what values should be included in the box you see when you hover over a data point on a scatter plot
- textab: write a latex table from a matlab table or array
